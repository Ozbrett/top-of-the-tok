# GitHub Pages site for Top Of The Tok

This folder is a simple static GitHub Pages site for the Chrome Web Store privacy policy/support URL.

## Before publishing

Replace every `REPLACE_WITH_SUPPORT_EMAIL` in:

- `privacy.html`
- `support.html`

with your public support email address.

## GitHub Pages setup

1. Create a new public GitHub repository, for example `top-of-the-tok`.
2. Upload these files to the repository root.
3. Go to repository Settings → Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Set Branch to `main` and folder to `/root`.
6. Save.
7. Wait a minute, then open the Pages URL.

Use the `privacy.html` URL as your Chrome Web Store privacy policy URL.
